Combien de sorte de DP sont cres par jour ?

ex. avec Slestat: on cre un objet de type liste form dautant de listes quil y a de jours (1 liste par jour). Chaque liste est forme par les codes CIM10 du jour, lesquels ont regroups par type grace la mthode table. Au final on obtient pour chaque jour la liste des codes CIM et pour chaque code, le nombre de dossiiers correspondants. Par la fonction length on compte le nombre de diagnostics uniques. Lensemble est rsum par la fonction summary.

sel <- d09[d09$FINESS == "Sel",]
x <- tapply(sel$DP, as.Date(sel$ENTREE), table)
# exemple:
length(x[[2]])
x[[2]]
# la fonction sapply permet de rcuprer un vecteur plutt qu'une liste.
m <- sapply(x, length)
summary(m)
